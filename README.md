# Bettiah Business Directory

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![GitHub Stars](https://img.shields.io/github/stars/SnakeEye-sudo/Bettiah-Business-Directory)](https://github.com/SnakeEye-sudo/Bettiah-Business-Directory/stargazers)

A crowd-sourced directory & review platform for local businesses in **Bettiah, West Champaran, Bihar**. Built with OpenStreetMap integration, user reviews, and listing submission capabilities. All data is stored locally/statically — no backend required.

## Features

- 🗺️ **OpenStreetMap Integration** — View businesses on an interactive map
- ⭐ **Reviews & Ratings** — Submit and browse community reviews
- 📋 **Business Listings** — Add your local business to the directory
- 📱 **Responsive Design** — Works on mobile and desktop
- 🔒 **No Backend Needed** — All data stored statically

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/SnakeEye-sudo/Bettiah-Business-Directory.git
cd Bettiah-Business-Directory
```

2. Open `index.html` in your browser — no build step required!
